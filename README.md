# 🌧️ Rainfall Prediction using Machine Learning

This project uses historical meteorological data to predict rainfall using various machine learning techniques. It includes data preprocessing, model training, and visual output of predicted rainfall.

## 📂 Dataset
The dataset includes historical records like:
- Rainfall in mm
- Date & location
- Temperature, humidity, etc.

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib / Seaborn
- Scikit-learn

## 📈 Output
- Predicted rainfall values
- Visualization graphs (bar chart, line plot)
- Accuracy score
